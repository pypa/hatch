# Security Policy

## Supported Versions

| Version  | Supported          |
| ---------| ------------------ |
| latest   | :white_check_mark: |
| <latest  | :x:                |

## Reporting a Vulnerability

Hatch and hatchling assume you have full control over your build environment. 

For potential security issues go to https://github.com/pypa/hatch/security/advisories/new

### DO NOT OPEN A PUBLIC ISSUE.
